- python-based telegram bot to control you PC
- For preventing any unwanted change in the bot, token was removed.
- This bot has abilities such as:
  file manager (Meaning you can download,delete and send file on you PC to you telegram chat)
  shutdown or restart
  taking screen shots
  playing media file (Download and play any media file)
  responding to simple sentences such as hello, how are you and ...
- PyTelegramBotAPI module was used to create the bot.
